In this introductory workshop, we will help you get started with Big Data Technologies.

Topics covered:
-------------------------
- Introduction to Big Data
- Big data use case sharing
- RDD Dataframe
- SAS to Hadoop for larger data

Prerequisite:
--------------------
- Basic Python knowledge
